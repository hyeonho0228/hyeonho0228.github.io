THRAISE & H³NGST: RNA-seq & ChIP-seq 자동 분석 플랫폼

🔬 THRAISE: RNA-seq 자동 분석

THRAISE는 high-throughput, high-resolution, hybrid RNA-seq 데이터를 자동으로 분석하는 플랫폼입니다.

FastQC, Trimmomatic, STAR, featureCounts, DESeq2 등을 활용해 빠른시간안에 논문용 분석을 완료하세요.

🔗 https://autornaseqhhh.duckdns.org/
(RNA-seq 자동 분석 플랫폼, 준비 중)

🧬 H³NGST: ChIP-seq 분석

H³NGST는 ChIP-seq 데이터를 자동으로 분석하는 플랫폼입니다.

peak calling부터 annotation, motif 분석까지 웹 기반으로 제공됩니다.

🔗 https://ngschiphhh.duckdns.org/
(ChIP-seq 분석 플랫폼, 서비스 중)

이 플랫폼은 대한민국의 생명과학 연구자들을 위하며, 순수히 연구 생산성 향상을 목표로 만들어낸 비상업적 프로젝트입니다.
더 많은 연구자들이 분석 장벽 없이 RNA-seq 및 ChIP-seq 데이터를 쉽게 처리하길 바라는 마음으로 제작되었습니다.

⛵ 오픈사이언스와 연구 재현성(reproducibility)을 지향합니다.
👋 만약 이 프로젝트에 관심이 있으시거나 제안을 원하신다면 언제든지 연락 주세요.
