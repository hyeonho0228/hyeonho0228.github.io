THRAISE & H³NGST: RNA-seq & ChIP-seq 자동 분석 플랫폼

🔬 THRAISE: RNA-seq 자동 분석

THRAISE는 high-throughput, high-resolution, hybrid RNA-seq 데이터를 자동으로 분석하는 플랫폼입니다.

FastQC, Trimmomatic, STAR, featureCounts, DESeq2, clusterProfiler, GSEA 등을 활용해 단 몇 분만에 논문용 분석을 완료하세요.

🔗 https://autornaseqhhh.duckdns.org/
(RNA-seq 자동 분석 플랫폼, 준비 중)

🧬 H³NGST: ChIP-seq / CUT&RUN 분석

H³NGST는 ChIP-seq 및 CUT&RUN 데이터를 자동으로 분석하는 플랫폼입니다.

BWA, MACS2, HOMER, SICER 기반으로 peak calling부터 annotation, motif 분석까지 웹 기반으로 제공됩니다.

🔗 https://ngschiphhh.duckdns.org/
(ChIP-seq 분석 플랫폼, 서비스 중)

본 플랫폼은 대한민국 생명정보학 연구자들의 연구 생산성 향상을 위해 만들어졌습니다.
